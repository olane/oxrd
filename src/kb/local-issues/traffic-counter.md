# Traffic counter

We have been kindly given a traffic counter by [Camcycle](https://www.camcycle.org.uk/). 
This is a camera based counter, but doesn't save any images. Instead, it uses the camera
to classify any passing vehicles, bikes and pedestrians, and stores only the counts, directions
and estimated speeds of each.

This provides valuable data to support our [rat running campaign](/kb/local-issues/traffic-petition/).

If you want to know more, including how to access the raw data for further analysis, contact Oli Lane.
